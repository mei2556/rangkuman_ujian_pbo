Part 01

Penanganan String

📝Apa itu String?

String adalah objek yang terdiri dari deretan karakter. Berbeda dengan tipe data primitif, panjang karakter String tidak perlu ditentukan dari awal.

Di Java, String dipandang sebagai objek, sehingga setiap data String memiliki method yang bisa dijalankan. Penggunaan method pada String inilah yang disebut operasi/penanganan String.

🔨Membuat String

Ada dua cara membuat String di Java:

// 1. Tanpa new (literal) — cara paling umum
String s1 = "Halo";

// 2. Dengan new (konstruktor)
String s2 = new String("Ini juga bisa");

// Keduanya menghasilkan String yang sama secara nilai

⚠️String null vs String Kosong

Keduanya berbeda dan sering membingungkan pemula:

Jenis Contoh Kode Artinya
String null s1 = null; Benar-benar tidak ada isinya — belum punya alokasi memori
String kosong s2 = ""; Ada isinya, tapi isinya kosong — sudah diinisialisasi, tidak punya karakter apapun

⚙️Method-Method Penting pada String

+ / concat()
Menggabungkan dua atau lebih String
length()
Menghitung jumlah karakter dalam sebuah String
charAt(index)
Mengambil satu karakter berdasarkan posisi/index (index mulai dari 0)
substring(a, b)
Mengambil sebagian karakter (dari index a sampai sebelum index b)
trim()
Menghapus spasi di awal dan akhir String
replace(lama, baru)
Mengganti kata/karakter lama dengan yang baru
toUpperCase()
Mengubah semua huruf menjadi huruf KAPITAL
toLowerCase()
Mengubah semua huruf menjadi huruf kecil
equals(str)
Membandingkan isi dua String (case-sensitive)
equalsIgnoreCase(str)
Membandingkan isi dua String tanpa memperhatikan besar/kecil huruf
contains(str)
Mengecek apakah sebuah String mengandung kata/karakter tertentu
public class ContohString {
  public static void main(String[] args) {
    String text = " Java Programming ";

    System.out.println("Original : " + text);
    System.out.println("Panjang  : " + text.length());      // 19
    System.out.println("Trimmed  : " + text.trim());       // "Java Programming"
    System.out.println("Upper    : " + text.toUpperCase()); // " JAVA PROGRAMMING "
    System.out.println("Lower    : " + text.toLowerCase()); // " java programming "
    System.out.println("Substring: " + text.substring(1, 5));  // "Java"
    System.out.println("Replace  : " + text.replace("Java", "Python")); // " Python Programming "
    System.out.println("Contains : " + text.contains("Java"));  // true
    System.out.println("Equals   : " + text.equals("JAVA"));    // false
  }
}
Part 02

Kondisi

💡Apa itu Kondisi?

Kondisi adalah keadaan yang menentukan kegiatan apa yang akan dilakukan, berdasarkan pernyataan yang sudah ada. Kondisi adalah sintaks yang membuat program bisa ber-LOGIKA.

Bersama perulangan, kondisi adalah sintaks yang paling sering digunakan di semua bahasa pemrograman. Java menyediakan dua cara: IF–ELSE dan SWITCH CASE.

🔀IF — ELSE : 4 Varian

Varian 1
IF saja

Digunakan untuk menjalankan kode hanya jika kondisi bernilai true.

Varian 2
IF — ELSE

Digunakan jika ada dua kemungkinan hasil yang berbeda berdasarkan kondisi tertentu.

Varian 3
IF — ELSE IF

Digunakan jika ada lebih dari dua kemungkinan kondisi. Namun lebih disarankan memakai Switch Case.

Varian 4
IF Bersarang (Nested)

Digunakan ketika perlu memeriksa kondisi di dalam kondisi lain (beberapa tingkat keputusan).

// Varian 1 — IF saja
if (angka > 5) {
  System.out.println("Angka lebih besar dari 5");
}

// Varian 2 — IF ELSE
if (angka > 5) {
  System.out.println("Lebih besar dari 5");
} else {
  System.out.println("Kurang dari atau sama dengan 5");
}

// Varian 3 — IF ELSE IF
if (nilai >= 90) {
  System.out.println("Nilai A");
} else if (nilai >= 75) {
  System.out.println("Nilai B");
} else {
  System.out.println("Nilai C");
}

// Varian 4 — IF Bersarang (contoh login)
if (username.equals("admin")) {
  if (password.equals("12345")) {
    System.out.println("Login berhasil!");
  } else {
    System.out.println("Password salah!");
  }
} else {
  System.out.println("Username tidak ditemukan!");
}

🔄Switch Case

Digunakan sebagai alternatif dari if-else if ketika ada banyak nilai yang perlu dibandingkan. Lebih rapi dan efisien untuk kondisi dengan banyak pilihan.

💡 Jangan lupa break; di setiap case! Tanpa break, program akan terus mengeksekusi case berikutnya (fall-through). Gunakan default: untuk menangani nilai yang tidak cocok dengan case manapun. 
// Sintaks dasar Switch Case
switch (ekspresi) {
  case nilai1:
    // kode jika ekspresi == nilai1
    break;
  case nilai2:
    // kode jika ekspresi == nilai2
    break;
  default:
    // kode jika tidak ada yang cocok
}

// Contoh: Menampilkan nama hari
int hari = 3;
switch (hari) {
  case 1: System.out.println("Senin");  break;
  case 2: System.out.println("Selasa"); break;
  case 3: System.out.println("Rabu");   break;  // ← output: Rabu
  default: System.out.println("Hari tidak valid");
}

📊Perbandingan: IF-ELSE vs Switch Case

Aspek IF — ELSE Switch Case
Cocok untuk Kondisi dengan rentang nilai / ekspresi kompleks Kondisi dengan banyak nilai spesifik yang dibandingkan
Operator logika Bisa (&& / || / !) Tidak bisa
Keterbacaan Menurun jika terlalu banyak else if Lebih rapi dan terstruktur untuk banyak pilihan
Tipe data ekspresi Semua tipe (boolean, int, String, dll) int, char, String, enum
Contoh use case Cek rentang nilai ujian (≥90, ≥75, dll) Menu pilihan, nama hari, zodiak, dll
