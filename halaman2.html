Variable & Tipe Data

Apa itu Variable?

Variable adalah tempat menyimpan data sementara yang bersifat dinamis — nilainya bisa diubah kapan saja sesuai kebutuhan.

💡 Analoginya seperti toples berlabel. Toples "Gula" hanya bisa diisi gula, toples "Garam" hanya garam. Label = Tipe Data, Toples = Variable.
Sintaks Deklarasi
// Format umum
tipe_data namaVariabel = nilai;

// Contoh
int angka = 10;
String nama = "Java";
boolean aktif = true;
Tanpa Inisialisasi
int x;
x = 100; // diisi nanti
Multiple Variable
int a=5, b=10, c=15;
Konstanta (final)
final double PI = 3.14159;
// PI tidak bisa diubah!
1.1

Tipe Data Primitif

⚠️ Prinsip Utama: Data hanya bisa dioperasikan jika memiliki tipe data yang sama. Contoh: 4 apel + 3 jeruk tidak bisa dijumlah secara langsung — sama seperti int + String akan menghasilkan ERROR.
Tipe Data Ukuran Rentang Nilai Contoh Penggunaan
byte 1 byte -128 s/d 127 byte umur = 25;
short 2 byte -32,768 s/d 32,767 short tahun = 2024;
int 4 byte ±2,147,483,647 int jumlah = 100000;
long 8 byte ±9.2 × 10¹⁸ long pop = 7800000000L;
float 4 byte ±3.4E-38 float pi = 3.14f;
double 8 byte ±1.7E-308 double phi = 3.14159265;
char 2 byte 0 s/d 65,535 char huruf = 'A';
boolean 1 bit true / false boolean aktif = true;
1.2

Tipe Data Non-Primitif (Referensi)

Berbentuk objek, memiliki method dan atribut. Dibuat oleh programmer atau berasal dari Java API.

String Array Class Datetime List Set Map Queue Integer Character Double

System.out.println(nama.charAt(0)); // J

int[] angka = {10, 20, 30};

02

Struktur I/O (Input / Output)

Java menyediakan beberapa cara untuk menangani operasi input dari pengguna dan output ke layar melalui paket java.io, java.nio, dan java.util.Scanner.

2.1

Output — System.out

Method Output
  • print() — Cetak tanpa pindah baris
  • println() — Cetak dengan pindah baris
  • printf() — Cetak dengan format string

Format String di printf():

%s → String %d → Integer %.2f → Float/Double \n → Newline
Contoh Kode
String nama = "Java";
int umur = 25;
double nilai = 95.567;

System.out.print("Tanpa Enter ");
System.out.println("- Dengan Enter");
System.out.printf(
  "Nama: %s, Umur: %d, Nilai: %.2f\n",
  nama, umur, nilai
);
2.2

Input — Scanner

Method Scanner
Method Membaca
nextInt() Angka bulat (integer)
nextDouble() Angka desimal
nextLine() String lengkap (satu baris)
nextBoolean() true / false
Contoh Kode
import java.util.Scanner;

Scanner input = new Scanner(System.in);

System.out.print("Masukkan nama: ");
String nama = input.nextLine();

System.out.print("Masukkan usia: ");
int usia = input.nextInt();

input.close(); // wajib ditutup!
2.3

BufferedReader & System.console()

BufferedReader
import java.io.BufferedReader;
import java.io.InputStreamReader;

BufferedReader reader = new
  BufferedReader(
    new InputStreamReader(System.in)
  );

String nama = reader.readLine();
int umur = Integer.parseInt(
  reader.readLine()
);

✓ Lebih cepat dari Scanner
✓ Efisien untuk banyak data
✗ Perlu konversi tipe manual

System.console()
java.io.Console c = System.console();
if (c == null) {
  System.out.println("N/A");
  return;
}

String nama = c.readLine("Nama: ");
char[] pass = c.readPassword(
  "Password: "
);

✓ Baca password (karakter tersembunyi)
✓ Lebih aman untuk input rahasia
✗ Tidak jalan di IDE (hanya CMD/terminal)

Metode Kecepatan Tipe Return Cocok untuk Bisa di IDE?
Scanner Sedang Langsung sesuai tipe Pemula, input sederhana ✓ Ya
BufferedReader ⚡ Cepat String (perlu konversi) Data banyak, performa tinggi ✓ Ya
System.console() Normal String / char[] Input password/rahasia ✗ Tidak
03

Operasi Dasar

3.1

Operasi Penugasan

Digunakan untuk memberikan nilai ke variable. Operator dasar: =

Integer & Double
int a = 10;
a += 5;  // a = a + 5  → 15
a -= 2;  // a = a - 2  → 13
a *= 3;  // a = a * 3  → 39
a /= 2;  // a = a / 2  → 19
a %= 4;  // a = a % 4  → 3
String & Boolean
// String
String text = "Hello";
text += " World!";
// Output: Hello World!

// Boolean
boolean d = true;
d = !d; // Negasi → false
3.2

Operasi Aritmatika

Untuk perhitungan matematika dasar. Output bertipe Numeric.

+Penjumlahan
-Pengurangan
*Perkalian
/Pembagian
%Modulus (sisa)
Contoh Dasar
int a = 10, b = 3;
System.out.println(a + b); // 13
System.out.println(a - b); // 7
System.out.println(a * b); // 30
System.out.println(a / b); // 3
System.out.println(a % b); // 1
Library Math

Untuk pangkat, akar, dll. gunakan class Math:

double hasil = Math.pow(2, 3);
// 2³ = 8.0

double akar = Math.sqrt(25);
// √25 = 5.0
3.3

Operasi Perbandingan

Membandingkan dua nilai. Output bertipe Boolean. Sering dipakai dalam kondisi if-else.

Operator Fungsi
== Sama dengan
!= Tidak sama dengan
> Lebih besar
< Lebih kecil
>= Lebih besar atau sama dengan
<= Lebih kecil atau sama dengan
Contoh dalam if-else
int angka = 10;

if (angka > 10) {
  System.out.println("Lebih besar");
} else if (angka == 10) {
  System.out.println("Sama dengan 10");
} else {
  System.out.println("Lebih kecil");
}
3.4

Operasi Logika

Operator Nama Fungsi
&& AND True jika kedua kondisi true
|| OR True jika salah satu kondisi true
! NOT Membalik nilai boolean
Contoh Kode
boolean a = true, b = false;

System.out.println(a && b); // false
System.out.println(a || b); // true
System.out.println(!a);     // false
3.5

Operasi Bitwise

Memanipulasi bit secara langsung. Digunakan dalam optimasi performa, kriptografi, pengolahan gambar, dan bit masking.

Operator Nama Contoh Penjelasan
& AND a & b Bit sama-sama 1 → 1, lainnya 0
| OR a | b Salah satu 1 → 1
^ XOR a ^ b Bit berbeda → 1, sama → 0
~ NOT ~a Membalik semua bit
<< Left Shift a << n Geser kiri n bit (= a × 2ⁿ)
>> Right Shift a >> n Geser kanan n bit (= a ÷ 2ⁿ)
>>> Unsigned Right Shift a >>> n Geser kanan tanpa mempertahankan tanda
3.6

Operasi String

Penggabungan string (concatenation) bisa dilakukan dengan + atau method .concat(). Operasi lain (seperti length(), charAt(), dll.) akan dibahas di pertemuan lain.

String a = "Java";
String b = "Programming";

String hasil1 = a + " " + b;
String hasil2 = a.concat(" ").concat(b);

// Keduanya menghasilkan: "Java Programming"

Contoh Program Lengkap — Kalkulator Sederhana
import java.util.Scanner;

public class Kalkulator {
  public static void main(String[] args) {
    Scanner input = new Scanner(System.in);

    System.out.print("Masukkan angka pertama: ");
    double a = input.nextDouble();

    System.out.print("Masukkan angka kedua: ");
    double b = input.nextDouble();

    System.out.print("Operator (+,-,*,/): ");
    String op = input.next();

    double hasil = 0;
    if     (op.equals("+")) hasil = a + b;
    else if (op.equals("-")) hasil = a - b;
    else if (op.equals("*")) hasil = a * b;
    else if (op.equals("/")) hasil = a / b;

    System.out.printf("Hasil: %.1f %s %.1f = %.1f\n",
      a, op, b, hasil);

    input.close();
  }
}
